# Interactive Personality Quiz Website

An interactive personality quiz website inspired by a PDF-based assessment shared during an internship training session with Inspiring Women. The project transforms the original manual result-calculation process into a more engaging and user-friendly digital experience.

## Features

* Interactive quiz interface
* Automated personality result calculation
* Responsive design for different screen sizes
* Personality descriptions and career mapping
* Clean and modern UI

## Built With

* HTML
* CSS
* JavaScript

## Live Website

https://inspiringwomenquiz.netlify.app/

## Project Inspiration

The idea for this project came from a discussion during an internship training session about making personality assessments easier and more interactive for users.
